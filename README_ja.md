# プライバシー カメラ by katahiromz

## これは何？

カメラ映像にリアルタイムで黒目線、ぼかし、黒塗りのいずれかを重ねて、写真撮影または録画ができるアプリです。

顔認識により、あなたのプライバシーを守る手伝いをします。

## 使い方

* 最初にアプリを起動すると、アプリがカメラとマイクの権限が要求しますので、許可してください。
* 自動的にカメラが起動し、認識された顔に黒目線、ぼかし、黒塗りのいずれかが描かれます。
* 下側の３つのボタンは次のような意味です。
    - カメラ切り替えボタン: 前面カメラ・背面カメラを切り替えます。
    - 写真撮影ボタン: 写真を撮影します。
    - 録画ボタン: 録画を開始または停止します。
* 左上の歯車は、設定ボタンです。アプリの設定ができます。

## 免責事項

- 現状では顔認識は確立されていない技術のため、常に正しく顔を認識することは原理的に不可能です。
- 明るすぎる場所、暗すぎる場所、日差しの当たった場所では顔認識に失敗することがあります。
- カメラの正面を向いていない顔、かたむいている顔、横になった顔は認識できないことがあります。
- 画面から一部または全部はみ出している顔は認識できないことがあります。
- 顔に何かを塗った顔、マスクをかぶっている顔、漫画やアニメの顔、動物の顔などは認識できないことがあります。
- はげしく動く顔、突然出てきた顔などは認識できないことがあります。
- 顔認識ができないときは、顔はそのまま表示されることがあります。
- 原理的に完全に顔を隠すことはできませんので、顔を完全に隠したい場合は、撮影後、映像を確認・編集してください。
